**TamgaOs (yula)

Experimental x86 operating system written in Zig and C.

This project started as a small learning kernel for exploring low-level systems programming, executable formats, bootloaders, and operating system internals.

For now the kernel only prints:

Zig -> KERNEL OK
C   -> KERNEL OK_C_

Both Zig and C versions successfully boot through Limine using a Multiboot2 header.
